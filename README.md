# 🏐 FuteVôlei — Sistema de Torneio

App de placar e gerenciamento de torneios de futevôlei.  
Funciona direto no navegador, sem instalação.

## 🔗 Acesso

> **[Abrir o App](https://SEU-USUARIO.github.io/futevolei)**

## ✨ Funcionalidades

- **Sistema de rodízio inteligente** por número de duplas
  - 6+ duplas: ganhou ou perdeu, sai — todos contra todos
  - 5 duplas: vencedor espera 1, perdedor espera 2
  - 3-4 duplas: vencedor fica 2× ou 3× seguidas
- **Placar fullscreen** para celular na horizontal
- **Ranking por jogador** — acumula stats de todas as duplas
- **Gerenciar duplas** — trocar jogadores, novas duplas, remover
- **Histórico** completo com duração de cada partida
- **Tela de resultados** com pódio, destaques e estatísticas do dia
- **localStorage** — salva automaticamente, resiste a travamentos
- **Data e nome do evento**

## 📱 Como usar no celular

1. Abra o link no Chrome ou Safari
2. Menu → **"Adicionar à tela inicial"**
3. O app aparece como ícone, funciona como app nativo
